  • O que é o JSON?

    O JavaScript Object Notation (JSON) é um formato de intercâmbio de dados, independente de linguagem, leve e baseado em texto. Foi derivado da linguagem de programação ECMAScript.

    JSON define um pequeno conjunto de regras de formatação para uma representação portável de dados estruturados.







  • Para que serve o JSON?



    Serve para possibilitar a troca de dados entre sistemas de forma simplificada e inteligível de forma leve e mais alinhada às necessidades dos programadores que precisam consumir webservices em seus projetos.







  • Qual a estrutura de dados do JSON?

    Apesar do JSON suportar objetos ele não é considerado uma linguagem de programação, mas pode ser (e é) um objeto da linguagem JavaScript e de outras linguagens.

    O JSON é baseado na simples representação literal de Objetos, Arrays, Strings, Números e Booleans do JavaScript

  • JSON X XML

    JSON é suportado pela maioria das linguagens modernas, e agora compete com o XML como protocolo de dados para serviços web, http e configuração de sistemas

  • XML 

    

<?xml version="1.0" encoding="UTF-8"?>
  <id>1</id>
  <nome>Alexandre Gama</nome>
  <endereco>R. Qualquer </endereco>

  • JSON 

    

{
	"id":1,
	"nome":"Alexandre Gama",
	"endereco":"R. Qualquer"
}





  • Vantagens do JSON

    
- Leitura mais simples
- Analisador (parsing) mais fácil
- Suporte à objetos
- Maior desempenho em velocidade de transporte de dados e execução
- Arquivo com tamanho reduzido
- Quem utiliza? Google, Facebook, Yahoo!, Twitter...
